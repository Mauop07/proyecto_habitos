# Proyecto de Gestión de Hábitos - Semana 2

Este repositorio contiene la solución dividida en Backend y Frontend.

## Estructura del Proyecto
- **/backend**: API construida con Express y MongoDB (Semana 1).
- **/frontend**: Aplicación web construida con Next.js y Redux (Semana 2).

## Instrucciones de Ejecución

### Backend
1. Entrar a `cd backend`.
2. Ejecutar `npm install`.
3. Configurar el archivo `.env`.
4. Ejecutar `node index.js`.

### Frontend
1. Entrar a `cd frontend`.
2. Ejecutar `npm install`.

3. Ejecutar `npm run dev` para iniciar Next.js.

## Semana 3: Integración de Tailwind CSS y UI Dinámica

En esta fase se implementaron los componentes visuales del frontend utilizando **Tailwind CSS**.

### Avances:
* **Tailwind CSS:** Configuración de la librería y uso de clases de utilidad para el diseño.
* **Barra de Progreso:** Componente visual que cambia de color (rojo a verde) basado en la meta de 66 días del libro *Hábitos Atómicos*.
* **Interfaz de Hábitos:** Lista dinámica con botones de acción ("Done") integrados.
* **Rama de Trabajo:** `semana3`
